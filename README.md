# 🧠 MUSLLY AI OS

## Healthcare Operating System for Yemen

**MUSLLY AI OS** هو منصة تشغيل صحية وطنية متكاملة، تبدأ من صيدلية المصلي في عدن وتتوسع لتغطية جميع محافظات الجمهورية اليمنية.

---

## 🌟 المميزات الرئيسية

- **☀️ AI SUN** — المخ الصناعي المركزي الذي يدير جميع العمليات
- **🪐 النظام الكوني** — 16 كوكب وظيفي مستقل يدور حول الشمس
- **🤖 وكلاء الذكاء** — 8+ وكلاء متخصصين (CEO, Medical, Inventory, Marketing, Finance, Doctor, WhatsApp, Vision)
- **📊 مركز القيادة** — Mission Control لمراقبة النظام في الوقت الفعلي
- **💬 محادثة ذكية** — محادثة مع AI مع اختيار الوكيل المناسب تلقائياً
- **🔒 أمان متكامل** — RLS, RBAC, Encryption, Audit Logs
- **🌙 الوضع الليلي** — دعم كامل للوضع الداكن

## 🚀 التشغيل

```bash
# تثبيت التبعيات
npm install

# تشغيل في وضع التطوير
npm run dev

# بناء للإنتاج
npm run build

# معاينة الإنتاج
npm run preview
```

## 🔧 التقنيات المستخدمة

- **React 18** + **TypeScript**
- **Vite 5** — Bundler
- **Tailwind CSS 3** — Styling
- **Framer Motion** — Animations
- **React Router DOM 6** — Routing
- **Zustand** — State Management
- **TanStack Query** — Data Fetching
- **Supabase** — Backend + Database
- **Lucide React** — Icons
